# termo-java
Termo Java
Uma implementação do jogo Termo desenvolvida em Java, inspirada no popular jogo de adivinhação de palavras. O projeto foi desenvolvido como parte dos meus estudos em Programação Orientada a Objetos e desenvolvimento de interfaces gráficas utilizando Java Swing.

Visão Geral
O objetivo do projeto é reproduzir a mecânica do jogo original, permitindo ao usuário descobrir uma palavra de cinco letras em até seis tentativas. A cada jogada, o sistema fornece um retorno visual indicando quais letras estão corretas, quais pertencem à palavra em posições diferentes e quais não fazem parte da resposta.

Principais recursos
Interface gráfica desenvolvida com Java Swing;
Geração aleatória da palavra a partir de um banco de palavras;
Validação de tentativas;
Tratamento de letras repetidas;
Teclado virtual integrado;
Sistema de vitória e derrota.
